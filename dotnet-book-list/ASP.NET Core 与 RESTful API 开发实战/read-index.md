## 媒体类型（Content Type，或MIME类型）的介绍

来源：第一章 REST简介；1.2.3 媒体类型；P7

用于指定资源的内容类型（Content Type），也称为媒体类型。

其语法为type/subtype，它由类型与子类型两个字符串构成，中间用“/”分隔，并且不允许空格存在。
