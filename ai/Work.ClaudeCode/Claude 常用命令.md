# Claude 常用命令

| 命令              | 示例 | 说明           |
| ----------------- | ---- | -------------- |
| claude            |      | 启动交互模式   |
| claude "任务描述" |      | 执行单次任务   |
| claude -p "问题"  |      | 快速查询后退出 |
| claude -c         |      | 继续最近的对话 |
| help              |      | 帮助           |
| Ctrl + C 或 exit  |      | 退出           |
