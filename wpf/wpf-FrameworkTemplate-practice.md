# FrameworkTemplate 派生类应用技巧

[FrameworkTemplate Class (System.Windows) | Microsoft Learn](https://learn.microsoft.com/zh-cn/dotnet/api/system.windows.frameworktemplate?view=windowsdesktop-9.0)

FrameworkTemplate 是一个抽象类，它是WPF中和模板相关的类型的父类，派生自 FrameworkTemplate 的类型有：

- [System.Windows.Controls.ControlTemplate](https://learn.microsoft.com/zh-cn/dotnet/api/system.windows.controls.controltemplate?view=windowsdesktop-6.0)
- [System.Windows.Controls.ItemsPanelTemplate](https://learn.microsoft.com/zh-cn/dotnet/api/system.windows.controls.itemspaneltemplate?view=windowsdesktop-6.0)
- [System.Windows.DataTemplate](https://learn.microsoft.com/zh-cn/dotnet/api/system.windows.datatemplate?view=windowsdesktop-6.0)
